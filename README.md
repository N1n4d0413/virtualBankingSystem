# Virtual Banking System

## 📌 Overview

This is a backend-based Virtual Banking System built using Spring Boot.
It supports basic banking operations such as user management, transactions, and admin control.

---

## 🚀 Features

* User & Admin authentication
* Deposit and fund transfer
* Transaction history
* Notification system (basic)
* Forgot password using security question

---

## 🛠 Tech Stack

* Java (Spring Boot)
* Maven
* MySQL (or compatible database)

---

## ▶️ How to Run

1. Install Java (JDK 17 or above)
2. Install IntelliJ IDEA (or any IDE)
3. Clone the repository
4. Open the project in IDE
5. Configure database in:

   ```
   src/main/resources/application.properties
   ```
6. Run `DemoApplication.java`
7. Open browser at:

   ```
   http://localhost:8080
   ```

---

## ⚠️ Note

* Update database credentials before running
* Project generated using Spring Initializr
* Some features are under improvement and may not cover all edge cases

---

## 📂 Project Structure

* `controller` → Handles API endpoints
* `models` → Entity classes
* `repos` → Database repositories
* `dto` → Data transfer objects
* `config` → Configuration classes
